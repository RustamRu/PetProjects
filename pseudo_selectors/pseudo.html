Псевдокласс :hover

Универсальный и наиболее часто используемый псевдокласс. Позволяет динамически изменять внешний вид элементов при наведении на них.

Пример

В блоке с классом "block10" при наведении на ссылку текст ссылки примет светло-зеленый цвет.

  3. Ссылка 3

}

Псевдоклассы :last-of-type, :first-of-type, :first-child, :last-child, :nth-child()

Работают очень похожим образом, выбирая либо первый либо второй элемент в контейнере. Отличие заключается в том, что для псевдоклассов last-of-type и first-of-type «первым» считается элемент, соответствующий тегу

Например: .container p:last-of-type{},

Выберет последний абзац в контейнере, даже если сам по себе этот абзац НЕ будет последним вложенным элементом

А например: .container p:last-child{}

Выберет последний элемент в контейнере, если он будет абзацем. Псевдоклассы :last-of-type и :first-of-type работают только совместно с селекторами типа!

Псевдокласс :nth-child() выбирает не первый или последний элемент (как предыдущие), а по порядковому номеру. Ознакомьтесь подробнее со спецификацией самостоятельно:

https://web-standards.ru/articles/nth-child/

Пример

В блоке с классом "block10" абзацы (элемент p) с псевдоклассами first-of-type/last-of-type выделены красным, причем первый из абзацев надчеркнут (из-за first-of-type), а последний - подчеркнут (из-за last-of-type) вне зависимости от его положения относительно элементов другого типа.

В блоке цитаты (элемент q) с псевдоклассами first-child/last-child выделены синим, причем первая из цитат не надчеркнута (из-за first-of-child), так как не является первый элементом, а последняя - подчеркнута (из-за last-child), так как является последним элементом в блоке.

В блоке пункты списка (элемент li) выделены оранжевым, причем те, чьи номера соответствуют результату вычисления по формуле 2n+1, где n варьируется от n до конца списка, т.е.:
1 (n=0, 2*0+1),
3 (n=1, 2*1+1),
5 (n=2, 2*2+1).

Псевдокласс :not()

Выбирает все элементы которые НЕ ТО, что записано в скобках.

Например: .container p:not(.article) {}

Выберет все абзацы, у которых НЕТ класса article, и которые находятся в контейнере с классом container

Пример

В блоке .block12 только те пункты списка будут выделены красным, которым НЕ присвоен класс .done

Псевдоэлементы ::before, ::after

Сейчас вам важно запомнить, что с помощью CSS можно так же динамически создавать дополнительные элементы в HTML разметке, что бывает очень удобно на практике. Например, для добавления иконок или небольших подписей. Для этого используются псевдоэлементы ::before и ::after

Пример

В блоке .block13 перед пунктами списка, для которых добавен псевдоэлемент :before, появится элемент с текстом "Сделано" со светло-зеленым фоном.
